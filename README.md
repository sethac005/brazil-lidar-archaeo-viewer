# Brazil LiDAR Archaeo Viewer

🚀 **Live Demo**: [https://sethac005.github.io/brazil-lidar-archaeo-viewer/](https://sethac005.github.io/brazil-lidar-archaeo-viewer/)

## Features
- 3D Cesium Globe with high-res imagery
- Hillshade/DSM layers with opacity
- Dynamic sun path, solstice presets, shadows for archaeology
- Point cloud integration (SP + Amazon placeholders)

## Usage
Open the link above. Customize by forking and editing index.html.

## Data Sources
- São Paulo: AWS Open Data
- Amazon: Embrapa + NASA ORNL

Add your processed tiles and enjoy!
